# 🎮 Simon Says Game

A simple memory game built using **HTML, CSS, and JavaScript**. The game generates a sequence of colors that the player must remember and repeat in the correct order. With each level, a new color is added to the sequence, making the game more challenging. If the player clicks the wrong color, the game ends and the final score is displayed.

## 🕹️ How to Play
- Press any key to start the game
- Watch the color that flashes
- Click the same color
- Repeat the full sequence as it grows each level
- The game ends if you press the wrong color

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
